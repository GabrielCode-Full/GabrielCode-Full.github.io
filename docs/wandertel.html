About the project

This project was created for my client, and my job is to change the current layout or UI based on the mockup provided by the client.

Wandertel is a hotel booking management system where guest can book a room in a hotel.

Features

  • The system has two user types: Administrators and Ordinary user
  • The user can sign in, sign up, change password, and forgot password.
  • The user can update the profile information.
  • The user can book a room and view it.

Platform Type

Web app

Technologies used

For front-end:

  • HTML
  • CSS
  • Javascript
  • Bootstrap

For back-end:

  • PHP

For database:

  • MySQL
