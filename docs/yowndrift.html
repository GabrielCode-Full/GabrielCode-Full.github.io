About the project

Yowndrift is a blog web application, where anyone can share their ideas or knowledge on a specific topic.

Features

  • The system has log in, register, reset password, and email verification.
  • In addition to typical, form based authentication, the system also used Laravel Socialite that supports authentication with GitHub and Google.
  • The users can also update account's profile information, email address, and password.
  • The system also provides two-factor authentication, browser sessions, and delete account option.
  • The system mainly uses the CRUD functions for creating, reading, updating, and deleting posts.
  • The users can also write comments to a post.
  • The system used Livewire for Front-end and Laravel for Back-end.
